# Sinemani

A DramaBox project for Tanzania.

## About

Sinemani is a streaming and entertainment platform tailored for Tanzanian audiences, delivering local and regional drama content.

## Features

- Browse and stream Tanzanian drama series
- Discover new local content
- User-friendly interface for all audiences

## Getting Started

Clone the repository and follow the setup instructions to get started with local development.

```bash
git clone https://github.com/magorikihore/sinemani.git
cd sinemani
```
